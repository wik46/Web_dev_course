This HTML looks at the 3 different positioning types.

It is used to change the positioning of the html elements.

  • Relative: relative to its default position.
  • Absoulte: relative to top left corner and other elements don't see it anymore.
  • Fixed: similar to absolute, but if user scrolls it stays in place.

This object is set with the highest z-index.

This object is moved using an absolute positioning.

Fixed positioning will work well with for a nav-bar or a button that you want on the screen. Also, because it has the highest z-index it will always be on top.

This division is only used so that we can scroll down and test the fixed positioning.
