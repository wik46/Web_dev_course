This HTML Margins and Borders.

They are used to add spaced to our divs.

Margins

margin 50 px constant.

This object is moved using an absolute positioning. That means that we can place this object anywhere and the other objects don't take it into consideration.

Margin 100px top and bottom, 50px left and right.

This division is only used so that we can scroll down and test the fixed positioning.

Padding

Very similar to margins. Just instead of a margin outside, we can have space inside the div.

With padding.

Note the space inside the div. Padding works very simliar to margins. You can remove the padding from this div element and see the difference. You can also add more padding to increase the distance of the text from the border.
Note that the padding adds to the width of the div.

No Padding

This is a div class with all the same properties as the one above. The only difference is that this one does no contain padding.

Note the following:

  • body, h1, p, ... have default margins and padding that you can remove.
  • Margin: the space around a div.
  • Padding: a "margin" but inside the div.
